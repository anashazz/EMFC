Get Started: Three Simple Steps

1.

Install Sensor Utility

Ensure your system has the Python `psutil` dependency installed for real-time temperature monitoring. This is a common requirement on many Linux systems.

sudo apt install python3-psutil
2.

Download the Executable

Download the PyInstaller-built standalone executable for Linux x64.

Download EMFC App (Linux x64)
3.

Run with Root Permissions

Make the file executable and run it. The application will handle the necessary password prompt for root access to the kernel module.

chmod +x EMFC_executable_name 
 ./EMFC_executable_name
